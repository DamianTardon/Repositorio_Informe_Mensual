    Se continuó el estudio de la norma IEC 60060-1, de la cual se tradujeron los puntos asociados al ensayo de impulso atmosférico.

    Se estudiaron los formatos de archivos posibles para almacenar las señales adquiridas, considerando criterios como eficiencia de almacenamiento y velocidad de lectura y escritura. A partir de este análisis, se seleccionaron los formatos BIN, H5 y CSV como los más adecuados para este proyecto. En particular, se empleará el formato BIN para respaldar los datos originales obtenidos del osciloscopio, el formato H5 para almacenar dichos datos y realizar el procesamiento digital de señales, y el formato CSV para la exportación de resultados.

    Se encuentra en desarrollo el módulo de administración de archivos, destinado a almacenar y gestionar las señales adquiridas. Actualmente, se dispone de un script que permite generar señales de prueba, guardarlas en los formatos seleccionados, y luego recuperarlas para graficarlas.

    Las actividades descritas en este informe se corresponden con las tareas 1 y 4 del diagrama de Gantt propuesto en la SAT.
